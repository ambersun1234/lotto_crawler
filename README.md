# lotto_crawler
crawler to get lotto history data([台灣彩券](https://www.taiwanlottery.com.tw/header.asp#))

## Prerequisite
```=1
make install
source ./env/bin/activate
sudo pip3 install -r ./requirements.txt
```

## Clone repo
```=1
git clone https://github.com/ambersun1234/lotto_crawler.git
```
